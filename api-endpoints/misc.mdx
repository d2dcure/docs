## All Misc. Endpoints in the Codebase

- 
- `contact.js` - Uses the nodemailer library to send emails to the d2dcure help email when a user submits the contact form 
- `report-bug.js` - Uses the nodemailer library to send emails to the d2dcure help email when a user submits the bug report form 
- `s3.ts` - Uses the s3 library from AWS to handle [cloud files storage](https://d2dcure.mintlify.app/the-basics/file-storage)
